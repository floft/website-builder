About Administration

To get to the administration page, you must first login. Go to yoursite.com/admin. On my site, I would go to http://garrett.floft.net/admin. Once you have arrived at the login screen, type in your username and password and click "Login." Below are all the pages you will be able to go to.

Menu
The menu will be on the left-hand side of the administration page. To go to a page, click on the link.

The Dashboard
You will now see "The Dashboard" page which will have a link to your website and it will tell you if there is a new version of the Floft Website Builder available.

Create a Page
This page will allow you to create a new page. Note: your homepage already exists.

Delete a Page
This page will allow you to delete the pages you have already created. Be careful when doing this; these pages can not be recovered.

Edit a Page
If you want to edit a page you have already created, click this link. You will be able to either use the TinyMCE editor or edit the HTML code for the page (to change this go to the "Edit Settings" page).

Edit the Links
This page will show you all the links on the menu bar and allow you to change the order. You will also be able to reset the order.

Edit the Footer
This page will show the current footer and allow you to edit it by using either the TinyMCE editor or by editing the HTML code.

Edit Error Page
This page will display the error page. This page will be displayed to a user when they try to access a page that does not exist. You will be able to edit this by using either the TinyMCE editor or by editing the HTML code directly.

View Uploads
This page will show you all of the files you have uploaded to your website. You will be able to delete, rename, and change whether or not they are executable.
Upload Something
This page allows you to upload more files to your website. If you want to upload a file larger than the set limit, you must edit the variables.php file and change the max upload size.

Pages only show if you have "Members" enabled on the "Themes" page:
Members
This page will show you all the members of your site, their username, name, and email address. It will also allow you to send a new user an activation email and allow you to delete a user.

Edit Member Links
This page is just like the "Edit the Links" page, but you will be editing the links for Members Only pages.

Edit Login Form
This will enable you to edit the text that is shown above the login form.

Create Account Text
This page shows you the text that is shown above the "Create an Account" form on the page. To view this, click the "View Page" link.
Themes
This page allows you to install and delete themes. Once you have installed a theme, you can select it and chose the options that go along with it.

Plugins
This page allows you to install and delete plugins. When you have plugins installed, you can click on the link (e.g. "Pictures") and edit the settings for that plugin, if any are available. Warning: plugins can do anything including delete your entire website, so be careful when you use plugins.

Site Stats
This page allows you to see how many people have gone to your website. To see how many people have gone to each individual page, click on the "View Stats" link next to the page.

Data Backup
Sometimes you might want to backup the databse. When you want to do this, go to this page and click the link to download a dump of your website content. Note: none of your uploads, themes, or plugins will be download.

Logout
This will log you out of the administration page. Use this when you are done editing/configuring your website.

My Account
You can edit your name, email address, username and password on this page if you ever want to change them.

Edit Settings
This page will allow you to change your settings. It will allow you to change whether you want to edit the pages using just plain HTML code or by using the TinyMCE (WYSIWYG, what you see is what you get) editor. Also, you can choose whether you want the default looking URLs or "Pretty URLs." Note: pretty URLs require Apache since it uses a .htaccess file.
