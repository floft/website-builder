Getting Started

Downloading
The first thing you need to do to get the Floft Website Builder is download it. You can do this by going to http://www.floft.net/webbuilder and downloading the "Website Builder Installer" which is the topmost link. Once you have downloaded this, open the zip file and copy its contents to the directory you would like.

Installing
Now, you need to go to your website and create a database to store your site. If you don't know how to do this, please read: http://www.siteground.com/tutorials/php-mysql/mysql_database_user.htm. After this, you need to add the database information into the variables.php file, so navigate to the directory you copied the files into and open variables.php. Fill in the information as follows:
$address = the URL to your site, like "garrett.floft.net"
$sitename = the name of your site (e.g. "Garrett")
$site_subtitle = what shows up under the title on some designs (the default)
$host_addon = the begining of the database name like if I have a database called "floft_garrett", this would be "floft"
$database = the rest of the database name, like the "garrett" part of "floft_garrett"
$main_username = the username for your database
$main_password = the password for your database
$max_upload_size = the max upload size (default is 500)
$site_id = this should remain zero (the default), otherwise you could encounter some problems
Then, open FileZilla and navigate to the directory you copied the files into (where the variables.php file is). Now upload them to your server. If you don't know how to do this, please read: http://wiki.filezilla-project.org/Using.

After you have uploaded all these files to the server, go to your website and you will see a beautiful website that has almost nothing on it. To change this and add the content you wish to see, go to your site address and type in /admin. For example, since my site is located on http://garrett.floft.net/, I would type in http://garrett.floft.net/admin. Note: you want to do this as soon as you get your files uploaded since whoever visits this address first can set up the administrative account.

Once you have logged in you will see a message telling you to update. Follow the instructions and you will be able to edit your website. Thank you for downloading the Floft Website Builder!

Problems
Getting data from database failed! - This means that it can't connect to the database. Make sure your database is set up and the variables in the variables.php file are correct.

You didn't add the tables into the database. Please do so. - This means that the tables weren't able to be inserted into the database. Import the database.sql (in this documentation directory) into the database. If you are using phpMyAdmin, click on the database you are using for your site (e.g. "garrett"). Once there, click on the "Import" tab at the top. Then, navigate to this directory and click the "Go" button. This should fix this problem.
