Blogger Plugin

Blogger feed url
An example is given next to this: http://example.blogspot.com/feeds/posts/default. Type this into the box if you are using Blogger for your blog and replace the example.blogspot.com with your username dot blogspot dot com.

Blogg feed url to link to
This can be any other feed link for example if you want a feedburner link, or this can be the same as the one above. You can also leave this blank if you don't want a link to the RSS Feed.

Subscribe to by email
This was created so there can be a link to the page FeedBurner uses to allow people to subscribe to your blog by email. If you don't use feedburner, this is pretty much useless. If you do, you can type in the email link here. It should look something like: http://feedburner.google.com/fb/a/mailverify?uri=floft/garrett&loc=en_US.
