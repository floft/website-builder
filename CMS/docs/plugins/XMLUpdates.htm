XML Updates Plugin

This plugin allows you to create a simple page listing the updates you have made to the site on certian days. Here is an example:
<updates>
     <day date="some date, some year">
          <update>updated blah blah blah</update>
          <update>another update blah blah</update>
     </day>
     <day date="some other date date, some other year">
          <update>updated blah blah blah</update>
          <update>another update blah blah</update>
     </day>
</updates>
You can add more days and ore updates. The page will display the date (the date attribute) in bold and the updates as bullet points below that.
